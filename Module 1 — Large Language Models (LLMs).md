# 📘 Module 1 — Large Language Models (LLMs)

I consider this the **most important technical module** in the entire course.

Why?

Because everything in modern AI Software Engineering revolves around LLMs.

* ChatGPT
* Claude
* Gemini
* DeepSeek
* Qwen
* Mistral
* GitHub Copilot
* Cursor
* Windsurf
* Claude Code
* Perplexity

All of them are powered by **Large Language Models (LLMs)**.

If you truly understand LLMs—not just how to use them, but how they work internally—you'll understand the foundation of modern AI.

---

# ⚠ Before We Start Module 1

I don't want to teach this module like a normal AI course.

Most courses explain:

> "LLMs predict the next token."

Then they move on.

That is **not enough**.

You should understand LLMs the same way a Computer Science student understands Operating Systems or Database Management Systems—not just the definition, but the internal architecture, algorithms, engineering trade-offs, limitations, and practical implications.

So I propose making **Module 1 the deepest module in the entire course**.

# 🎯 Module 1 Learning Outcome

By the end of Module 1, you should be able to:

* Explain how an LLM works internally from first principles.
* Understand why transformers revolutionized AI.
* Reason about tokens, context windows, embeddings, and attention.
* Compare modern LLM families and choose the right model for a task.
* Understand the costs, limitations, and engineering trade-offs of using LLMs.
* Build a strong conceptual foundation for all later modules, including Prompt Engineering, Context Engineering, RAG, AI Agents, and MCP.
